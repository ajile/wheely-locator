Здесь должны быть расположены скрипты, возможно с использование предпроцессора по типу: `coffee`, `typescript` и т.п.
Далее эти файлы преобразуются в JS, частично объединяются, минифицируются, аглифицируются и кладутся в директорию неподалеку `/public/javascripts`, которая в дальнейшем разлетается по CDN серверам и кормится пользователям.

Файлы из этой директории могут быть использованы на дев. версии приложения.
