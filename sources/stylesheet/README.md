Здесь должны быть расположены файлы описывающие стили, возможно с использование предпроцессора по типу: `less`, `sass`, `stylus` и т.п.
Далее эти файлы преобразуются в CSS, объединяются, минифицируются и кладутся в директорию неподалеку `/public/stylesheets`,
которая в дальнейшем разлетается по CDN серверам и кормится пользователям.

Файлы из этой директории могут быть использованы на дев. версии приложения.
